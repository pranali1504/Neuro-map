🧠 NeuroMap – Secure Full-Stack Web Application

📌 Overview

NeuroMap is a secure full-stack web application built using **Spring Boot MVC architecture**. The application provides robust authentication, role-based access control, and efficient backend services integrated with a relational database.

🚀 Features

 🔐 JWT-based Authentication & Authorization
 👥 Role-Based Access Control (RBAC)
 🌐 RESTful API Development
 🗄️ MySQL Database Integration (JPA/Hibernate)
 ⚙️ Maven-based Dependency Management
 🧪 API Testing using Postman
 ⚠️ Global Exception Handling
 🎨 Interactive Frontend (HTML, CSS, JavaScript)

🏗️ Tech Stack

🔹 Backend

* Java (Spring Boot, Spring MVC)
* Spring Security (JWT Authentication)
* JPA / Hibernate
* Maven

🔹 Frontend

* HTML
* CSS
* JavaScript

🔹 Database

* MySQL

⚙️ How to Run the Project

🔹 Backend (Spring Boot)

1. Open project in Eclipse or IntelliJ
2. Configure MySQL database in `application.properties`
3. Run:
mvn spring-boot:run

🔹 Frontend

1. Open `index.html` in browser
2. Or run using Live Server (VS Code)


 🔐 Authentication Flow

1. User registers / logs in
2. Server generates JWT token
3. Token is used for accessing protected APIs
4. Role-based authorization controls access

🧪 API Testing

* Tested using **Postman**
* Includes endpoints for:

  * Authentication
  * User management
  * Secured resources


 📈 Key Highlights

* Secure backend using JWT & Spring Security
* Clean MVC architecture
* Scalable REST API design
* Full-stack integration
